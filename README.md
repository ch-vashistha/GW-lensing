# GW-lensing

1. Gravitational wave propagation beyond geometric optics
   
   Reference: Giulia Cusin and Macarena Lagos. Gravitational wave propagation beyond geometric optics. Physical Review D, 101(4), February 2020

   Mathematica notebook implementing derivations related to the perturbative expansion of gravitational wave propagation beyond the geometric optics limit,             following the framework of Cusin & Lagos (2020). The notebook reproduces selected analytical steps related to the transport equations, polarization structure,
   and first-order corrections arising from diffraction effects.
